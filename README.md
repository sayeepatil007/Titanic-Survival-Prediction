# 🚢 Titanic Survival Prediction

## 📌 Project Overview
This project predicts whether a passenger survived or not using machine learning techniques. It involves data cleaning, exploratory data analysis (EDA), and building a classification model.

---

## 📊 Dataset
- Dataset used from Kaggle Titanic Competition  
- Contains passenger details like Age, Sex, Fare, Class, etc.

---

## 🧠 Features Used
- Pclass
- Sex
- Age
- Fare
- Embarked

---

## ⚙️ Steps Performed
- Data Cleaning (handled missing values)
- Exploratory Data Analysis (EDA)
- Feature Engineering (encoding categorical data)
- Model Building using Logistic Regression
- Model Evaluation using Accuracy Score

---

## 📈 Model Performance
- Achieved **~81% accuracy** on test data

---

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-learn
